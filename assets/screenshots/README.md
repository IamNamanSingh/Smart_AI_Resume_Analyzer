# Application Screenshots Directory

This directory contains the widescreen screenshots used to document the Smart Resume AI platform interface.

## Required Files

Please save the screenshots captured at `1920x1080` resolution using the following filenames:

1. `home.png` - Home page view.
2. `analyzer.png` - Resume Analyzer tab scorecard view.
3. `builder.png` - Resume Builder wizard form steps view.
4. `dashboard.png` - Admin Analytics Dashboard and interactive charts.
5. `feedback.png` - Rebranded category feedback cards.
6. `about.png` - Creator profiles card layout.
7. `admin.png` - Password login screen for metrics panel.

*(See [SCREENSHOT_GUIDE.md](../../SCREENSHOT_GUIDE.md) in the root directory for capturing instructions.)*
