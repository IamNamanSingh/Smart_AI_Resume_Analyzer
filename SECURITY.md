## Reporting a Vulnerability

We take the security of Smart Resume AI seriously. If you find a security vulnerability, please report it immediately:

1. **Do not open a public GitHub issue** for security bugs.
2. Email your report directly to the maintainer:
   **namansingh2475@gmail.com**
3. Include detailed reproduction steps, potential impact descriptions, and system versions.

You can expect an initial response acknowledging your report within 48 hours, along with updates on remediation progress.
